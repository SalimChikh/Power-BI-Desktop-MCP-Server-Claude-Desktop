# Power-BI-Desktop-MCP-Server-Claude-Desktop

## **Installation de Claude Desktop avec le package `.mcpb` (Recommandé)**

Le package `.mcpb` est la manière la plus simple d’installer l’extension : il s’agit d’un module autonome spécialement conçu pour Claude Desktop.

---

## **1. Téléchargement du package**

**Version complète :**  
`powerbi-desktop-mcp-2.0.10.mcpb`  
Inclut tous les outils : création, mise à jour, suppression.

**Version en lecture seule :**  
`powerbi-desktop-mcp-readonly-2.0.10.mcpb`  
Permet uniquement la consultation et l’analyse.

---

## **2. Installation de l’extension**

### **Méthode simple**

Double-cliquez sur le fichier `.mcpb`.  
Claude Desktop s’ouvrira automatiquement et affichera la fenêtre d’installation.

### **Méthode alternative**

1. Ouvrez **Claude Desktop**
2. Allez dans **Paramètres** (icône d’engrenage)
3. Ouvrez **Extensions**
4. Cliquez sur **Paramètres avancés**
5. Ouvrez la section **Extension Developer**
6. Cliquez sur **Installer une extension…**
7. Sélectionnez le fichier `.mcpb` téléchargé
8. Cliquez sur **Installer**

---

## **3. Redémarrage de Claude Desktop**

Quittez complètement l’application puis relancez-la  
(ne vous contentez pas de fermer la fenêtre).

Après redémarrage, le serveur sera automatiquement configuré et prêt à l’emploi.  
Un indicateur **MCP** apparaîtra en bas à droite de la zone de saisie lorsque le serveur est actif.
